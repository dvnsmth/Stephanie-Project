# Docs

This folder separates **mechanics** (execution contracts) from **policy** (creative/taste constraints).

- Start here: `MECHANICS.md`
- Policy layer: `POLICY.md`
- Provider wiring contract: `PROVIDER_CONTRACT.md`

Rule of thumb:
- If it changes filenames, headings, gates, or run state → **mechanics**.
- If it changes what is allowed/funny/appropriate → **policy**.
